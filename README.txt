BERNARDI CONSTRUCCIONES SRL - WEB

Cómo usar este proyecto:

1) Descomprimir el ZIP.
2) Abrir la carpeta en VS Code.
3) En la terminal ejecutar:
   npm install
4) Para verla local:
   npm run dev
5) Para subir a Vercel:
   - subir esta carpeta a GitHub
   - importar el repositorio desde Vercel
   - Vercel detecta Vite automáticamente

Contenido principal:
- src/App.jsx = diseño de la web
- src/main.jsx = arranque de React
- index.html = base del sitio
- package.json = dependencias
